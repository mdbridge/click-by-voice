Elements made focusable via contenteditable

Normally, a plain <div> or <span> is not focusable. Adding contenteditable makes it an editable region that can receive focus and caret.

Elements made focusable via tabindex

The tabindex attribute controls whether an element is focusable, and where it appears in the tab order.

  • tabindex="0" → focusable, participates in normal tab order
  • tabindex=">0" → focusable, custom tab order (discouraged)
  • tabindex="-1" → focusable by script only (no tab stop)

Try tabbing through the controls below and watch the order.

<div tabindex="0"> — focusable, in normal tab order
<div tabindex="2"> — focusable, with positive tabindex (comes earlier in tab order than some elements)
<div tabindex="1"> — focusable, custom tab order (will usually get focus before tabindex="2" and 0)
<div tabindex="-1"> — focusable via script only; you cannot tab to this.
